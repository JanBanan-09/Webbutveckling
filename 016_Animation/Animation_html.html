Lär dig CSS-animationer

I den här sidan kommer vi steg för steg lära oss hur man kan animera element med CSS. Vi använder @keyframes för att skapa rörelser och färgändringar.

Vad är CSS-animationer?

CSS-animationer låter oss förändra utseendet på ett element över tid. Istället för att sidan bara ser statisk ut kan element röra på sig, ändra färg, storlek eller position. Det finns två huvudmetoder: Transitions och @keyframes. Här fokuserar vi på @keyframes eftersom det ger mer kontroll.

Hur fungerar @keyframes?

@keyframes låter oss definiera steg för steg hur ett element ska ändras under animationen. Man skriver procenttal mellan 0% och 100% där varje steg beskriver hur elementet ska se ut just där i animationen.

Exempel på en enkel animation:

@keyframes myAnimation {
  0%   { background-color:red; left:0px; top:0px; }
  25%  { background-color:yellow; left:200px; top:0px; }
  50%  { background-color:blue; left:200px; top:200px; }
  75%  { background-color:green; left:0px; top:200px; }
  100% { background-color:red; left:0px; top:0px; }
}

Här händer följande:

  • 0% – Animationen börjar, boxen är röd uppe till vänster.
  • 25% – Boxen blir gul och flyttar sig till höger.
  • 50% – Boxen blir blå och flyttar neråt.
  • 75% – Boxen blir grön och flyttar till vänster.
  • 100% – Boxen är tillbaka till startläge och röd igen.

För att animationen ska synas måste vi koppla den till ett element:

div {
  width: 100px;           /* boxens bredd */
  height: 100px;          /* boxens höjd */
  position: relative;      /* viktigt för left/top */
  background-color: red;  /* startfärg */
  animation-name: myAnimation;  /* vilken animation som används */
  animation-duration: 4s;       /* hur lång tid ett varv tar */
  animation-iteration-count: infinite; /* loopar oändligt */
}

Förklaring steg för steg:

  • width & height – Hur stor boxen är.
  • position: relative – Gör det möjligt att flytta boxen med left och top.
  • background-color – Startfärg på boxen.
  • animation-name – Kopplar boxen till @keyframes.
  • animation-duration – Bestämmer hur lång tid ett varv tar.
  • animation-iteration-count – Loopar animationen oändligt.
